# EV2: Earth Version 2

## Overview

**EV2** is a free-to-play sci-fi looter-shooter RPG developed by Funtico and Frozen Dawn Entertainment, available on PC (Steam), PS5, and Xbox. Players fight in one of five **EV2 combat suits** (Brute, Cloaker, Pathfinder, Mag, Valkyrie) against an interdimensional infestation, looting enemies for weapons and gear while engaging in PvE campaigns, raids, or competitive PvP modes.

**Blockchain Focus**: EV2 integrates Web3 technology through tradeable NFT assets (weapon components, gear pieces, cosmetics) on Avalanche C-Chain, enabling true asset ownership and a player-driven economy with play-to-earn opportunities.

---

## Web3 Integration

EV2 leverages **Avalanche C-Chain (AVAX)** for blockchain integration, enabling:

### Currency System
- **Holocrons** (Soft): Earned free through gameplay for Common/Uncommon/Rare NFTs
- **EV2 Tokens** (Hard): Purchased or won in tournaments for Epic/Legendary/Exotic NFTs, battle passes, and cosmetics

### NFT Assets (Tradeable)
- **Weapon Components**: 6 modular parts (bullets, barrels, triggers, mags, stocks, scopes) crafted into custom weapons
- **Gear Pieces**: 5 armor slots (helmet, torso, arms, legs, jetpack) with stat bonuses
- **Chroma Skins**: Visual cosmetics for suits and weapons
- **Access Tokens**: Season/content access passes
- **Guild Tokens**: DAO-based organization management

### Play-to-Earn
- **Apex Trials**: 3v3 PvP with EV2 token buy-ins and rewards
- **High-Stakes PvE**: Select raids and strikes with token rewards
- **Secondary Market**: All NFTs tradeable on blockchain marketplaces

## Technical Details

### Platforms & Status
- **Release Date**: To be announced
- **Platform**: PC (Steam), PS5, Xbox
- **Genre**: Looter-Shooter, Free-to-Play, Web3 Gaming

### Blockchain Specifications
- **Network**: Avalanche C-Chain (AVAX)
- **Standards**: EIP-721 (NFTs), EIP-998 (Composable), EIP-1155 (Multi-Token), EIP-2535 (Diamond Pattern)
- **Wallet**: Non-custodial via WEB3S plugin with PlayFab cloud backup

### System Requirements
**Minimum**: Windows 10/11, Intel i5-4790, 8GB RAM, GTX 1060, 13GB storage  
**Recommended**: Intel i7-4790 / AMD Ryzen 3 1300X, 16GB RAM, GTX 1060/RTX 2060, 13GB storage
