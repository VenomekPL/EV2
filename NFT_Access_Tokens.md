# Access Tokens NFT

## Overview

Access Tokens are purchasable NFTs that grant account-wide access to exclusive content, specialized events, and instanced dungeons. These tokens serve as keys to progressively unlocked content tiers, with options ranging from one-time activities to permanent long-term access, each potentially featuring embedded progression systems.

---

## Token Types

### Early Access Tokens

Grants access to game content before general availability:

- **Early Access to Full Game**: Access to the main game during closed or early access periods
- **Beta Features**: Testing of new systems and balance changes
- **Advance Availability**: Access to new content patches and expansions
- **Priority Onboarding**: VIP newcomer support and resources
- **Permanent Account Unlock**: Applied account-wide to player's account

### Event Access Tokens

Grants participation in limited-time or exclusive events:

- **Tournament Access**: Entry to competitive events and leaderboard seasons
- **Seasonal Events**: Participation in time-limited story campaigns
- **Special Quests**: Exclusive mission chains and storylines
- **Collaboration Events**: Partner crossover events and sponsored campaigns
- **Transferable**: Can be gifted or traded to other players

---

## Limited-Use Content Access

### Relic Locations (Example Limited-Use Token)

Specialized tokens granting access to instanced dungeons with enhanced rewards:

#### Mechanics

- **Instanced Dungeons**: Private instance created specifically for authorized players
- **Enhanced Loot Chance**: Better than average loot rates compared to open-world content
- **Limited Approaches**: Token grants finite number of runs (e.g., 5 runs per week)
- **Team Requirements**: Dungeons sized for specific team compositions:
  - Solo dungeons (1 player)
  - Squad dungeons (3-4 players)
  - Raid dungeons (6-8 players)
- **Recharge Mechanics**: Uses replenish on a timer (daily, weekly, or seasonal)

#### Example Token Variants

- **Relic Location: Sunken Temple** - 5 runs per week, squad of 4
- **Relic Location: Ancient Vault** - 3 runs per week, solo or duo
- **Relic Location: Hive Nest** - 2 runs per week, raid team of 8

### Single-Use Access Tokens

Limited-time or one-time use tokens:

- **Challenge Mode Access**: One-time attempt at ultra-difficulty dungeons
- **Exclusive Raid Lockout**: Guaranteed raid access for limited period
- **Special Event Entry**: One-time participation in exclusive activities
- **Consumable Nature**: Token consumed upon use

## Long-Term Access Content (LDC)

### Persistent Access Tokens

Permanent or extended-duration tokens granting ongoing content access:

#### Mechanics

- **DLC-Style Content**: Permanent additional content added to account
- **Indefinite Access**: Not limited by number of uses
- **Account-Bound or Transferable**: Player choice determines if token stays with account or can be gifted
- **Progression Embedded**: Tokens include progression systems
- **Persistent Rewards**: Ongoing benefits while token is owned

### Embedded Progression Systems

Access tokens can include progression tracks:

#### Experience & Unlocks

- **Experience Points**: Earn XP while utilizing the access token
- **Progression Tiers**: Unlock additional rewards through cumulative XP
- **Milestone Rewards**: Special items, cosmetics, or currency at tier milestones
- **Season-Based Progression**: Reset and rotate progression seasons for engagement
- **Accumulation**: Experience carries over across multiple dungeon runs

#### Example: DLC Armor Set Access

- **Token**: "Void Armor Collection DLC"
- **Base Access**: Permanently unlocks all Void Armor patterns on account
- **Progression Track**: Gather Void Shards from missions while owning token
- **Tier 1** (100 Void Shards): Unlock Void Armor aesthetic customization
- **Tier 2** (250 Void Shards): Unlock exclusive Void Armor cosmetic variant
- **Tier 3** (500 Void Shards): Unlock bonus stat boosts for Void Armor
- **Tier 4** (1000 Void Shards): Unlock mythic tier Void Armor skin

## Acquisition & Transferability

### Primary Purchase
- Buy access tokens directly from EV2 marketplace
- Permanent tokens provide long-term value proposition
- Limited-use tokens provide recurring value

### In-Game Acquisition
- Earn tokens through gameplay achievements
- Complete challenging mission chains for token rewards
- Event participation grants temporary tokens
- Battle pass rewards include access token tiers

### Transfer & Gifting

#### Account-Bound Tokens
- Applied exclusively to purchasing account
- Cannot be traded or gifted
- Typically: permanent content unlocks and season passes

#### Transferable Tokens
- Can be traded on secondary market
- Can be gifted to other players
- Player retains control of distribution
- See [Second-Hand Economy](Second-Hand Economy.md) for marketplace details

#### Time-Limited Transfers
- Some tokens expire after set duration
- Transfer not possible after expiration
- Resets if transferred to new owner
- Creates seasonal collection incentive

## Progression Integration

### Experience Accumulation

- Active token holders gain XP while playing content
- XP accrues toward progression tiers
- Different content grants different XP rates
- Tracking on-chain for transparency

### Reward Structure

- Base rewards for token ownership
- Milestone bonuses at tier completion
- Seasonal progression resets
- New reward tiers added with token updates

### Player Engagement

- Ongoing reasons to use owned tokens
- Long-term progression goals
- Seasonal collection incentive
- Completionist appeal

## NFT Specifications

- **Asset Type**: Purchasable NFT (also obtainable via in-game rewards)
- **Tradeable**: Yes for time-limited and long-term tokens (varies by token type)
- **Blockchain Verified**: Ownership, uses, and progression recorded on-chain
- **Expiration**: Some tokens have automatic expiration dates
- **Use Tracking**: Remaining uses/availability tracked on-chain

## Content Access Model Benefits

### Player Retention
- Long-term progression keeps players engaged
- Embedded progression gives continuous goals
- Event-driven tokens create recurring spending
- Varied token types appeal to different playstyles

### Revenue Stability
- Season passes create predictable revenue
- Limited-use tokens encourage purchases
- Transferable tokens enable secondary market
- DLC-style tokens justify long-term spending

### Fairness & Accessibility
- In-game earning paths reduce pay-to-win perception
- Progression tracks transparent and observable
- Gifting enables community engagement
- Secondary market provides acquisition alternatives
